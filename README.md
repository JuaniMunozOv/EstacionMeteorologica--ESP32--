# Estación Meteorológica con ESP32 y ThingSpeak

Descripción:

La Estación Meteorológica con ESP32 y ThingSpeak es un proyecto que combina la potencia del ESP32, sensores de temperatura y humedad, y la plataforma en la nube ThingSpeak para crear una solución completa de monitoreo climático. Esta estación captura datos ambientales, como la temperatura, la humedad y el índice de calor, y los envía a ThingSpeak para su almacenamiento y visualización en tiempo real.

Características Destacadas:

Uso de sensores DS18B20 y DHT11 para medir la temperatura y la humedad del entorno.
Comunicación con la plataforma ThingSpeak a través de conexiones HTTPS para almacenar y visualizar los datos.
Conexión a redes WiFi mediante el ESP32 para permitir la configuración y transferencia de datos.
Manejo de errores y retroalimentación visual mediante indicadores LED.
Integración de una página web interactiva para configurar la red WiFi de la estación meteorológica.
Proyecto de código abierto alojado en GitHub para compartir, colaborar y mejorar.

Componentes y Tecnologías Utilizadas:

ESP32: Plataforma de desarrollo de bajo costo y alto rendimiento.
Sensores DS18B20 y DHT11: Para medir la temperatura y la humedad respectivamente.
ThingSpeak: Plataforma en la nube para el almacenamiento y la visualización de datos.
WiFi y HTTPS: Comunicación inalámbrica y segura para la transmisión de datos.
HTML y JavaScript: Creación de una página web interactiva para la configuración de red WiFi.
GitHub: Plataforma de alojamiento de código fuente y colaboración en desarrollo.
